# UISU Tutorials System: Roadmap & Feature Suggestions

This document outlines 12 proposed features and "edges" (innovative differentiators) to enhance the Tutorials ecosystem.

## Core Feature Enhancements

1.  **AI-Powered Quiz Generator**
    *   *Concept:* Automatically generate multiple-choice quizzes from the content of "Text" and "Essay" modules using LLMs.
    *   *Benefit:* Reduces tutor workload and provides immediate assessment for students.

2.  **Gamification & Academic Streaks**
    *   *Concept:* Implement a point system ("Aluta Points") for completing modules and maintaining daily study streaks. Unlockable badges for specific achievements (e.g., "Night Owl" for late-night study).
    *   *Benefit:* Increases student retention and daily engagement.

3.  **Interactive Code Playground**
    *   *Concept:* Embed a code editor (like Monaco Editor) within "Tech" tutorials, allowing students to run Python/JS code directly in the browser.
    *   *Benefit:* Practical application for CS/Engineering students without leaving the platform.

4.  **Voice Q&A for Tutors**
    *   *Concept:* Allow students to record short voice notes to ask questions, and tutors to reply with voice.
    *   *Benefit:* More personal connection and easier for complex explanations than typing.

5.  **Smart Flashcards Mode**
    *   *Concept:* "Focus Mode" that converts tutorial summaries into spaced-repetition flashcards.
    *   *Benefit:* Aids long-term retention of material.

6.  **Collaborative Class Notes**
    *   *Concept:* A wiki-style notes section attached to every tutorial where Verified Students can contribute and refine shared knowledge.
    *   *Benefit:* Crowdsourced knowledge base.

## "Edges" (Differentiating Innovations)

7.  **Offline-First Peer-to-Peer Sharing**
    *   *Concept:* Leverage the existing `sql.js` architecture to allow students to transfer downloaded tutorials to peers via local WiFi/WebRTC without using internet data.
    *   *Edge:* Critical for a campus environment where internet access may be unstable or expensive.

8.  **Co-Curricular Transcript Sync**
    *   *Concept:* Automatically generate an official "UISU Skills Transcript" PDF at the end of a session, listing all completed tutorials and verified skills.
    *   *Edge:* Transforms casual learning into a tangible credential for CVs.

9.  **Alumni Masterclass Vault**
    *   *Concept:* Exclusive, high-value tutorials created by successful UI Alumni, accessible only to students in specific levels (e.g., Final Year).
    *   *Edge:* leveraging the university's strong alumni network for mentorship.

10. **"Study With Me" Live Rooms**
    *   *Concept:* Virtual study rooms with lo-fi music and Pomodoro timers where students enrolled in the same course can see who else is online (presence indicators).
    *   *Edge:* Recreates the social pressure/motivation of a physical library digitally.

11. **Dynamic Syllabus Mapping**
    *   *Concept:* Map tutorials directly to specific University of Ibadan course codes (e.g., "GES 101"). Users subscribe to their "Course List" and get notified when content drops for their specific classes.
    *   *Edge:* Hyper-localization makes the platform indispensable for academic success.

12. **The "Aluta" Protocol (Micro-bounties)**
    *   *Concept:* Students can request a specific tutorial (e.g., "Explain MTH 201 Vector Spaces") and pledge points. Tutors earn "Bounties" by fulfilling these specific requests.
    *   *Edge:* Demand-driven content creation ensures the library is always relevant.
